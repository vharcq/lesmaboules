Tentaive de gestion des bénévoles dans l'oganisation d'une festival de marionette appellé MaBoule à Genappe.  

## Dev

Il faut developper un site Web en peu importe la techno mais tout doit etre dans le cloud

Pour les gestion du mail, si on ne sait pas SMTP alors eventuellement fair un lien vers le gmail voire utiliser gmail pour se logger et pouvoir envoyer les mails via ceci



## BackOffice

dit etre gérer avec un bete login/mot de passe unique pour un admin (ou 2)

Il s'agit de répertoirié:
- les benevoles (GSM/email)
- les salles de spectacles
- les spectacles
-   comapgnie,
-   origine (pays/ville),
-   nombre de personnes,
-   durée,
-   durée prepa et cloture
- de placer manuellement des spectacles sur un agenda de 5 jours max : "représentation"
- gerer les disponibilités des benevoles par demi journée
- associer les benevoles aux spectacles
-   besoin de 2 ou 3 personnes : avoir un defaut par salle de spectacle et associer sur une réprésentation

## Front Office

- ENvoyer des mails aux benevoles pour les prevenir de ou ils devront etre à quelle heure (voir delai de preparation sur un spectacle)
- gérer les modifications de dernietre minutes
- essayer de repartir equitablement

C'est tout pour commencer
